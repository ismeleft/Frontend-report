## 備註

圖片中，除了生活照，其餘皆來自 unsplash。

## 安裝和運行

1. 複製儲存庫：
   ```
   git clone https://github.com/your-username/yings-corner.git
   ```

2. 進入專案目錄：
   ```
   cd yings-corner
   ```

3. 安裝依賴：
   ```
   npm install
   ```

4. 運行開發伺服器：
   ```
   npm run dev
   ```

5. 在瀏覽器中開啟 `http://localhost:5173` 查看網站。
